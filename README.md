# qa_robotframework

Step 1: Get geckodriver and put it in the venv bin/Scripts.

## How to run robot framework: 
robot {filename}.robot

## How to run virtial environment (venv):

Windows:
python -m venv venv
.\venv\Scripts\activate 

Mac:
python3 -m venv venv
source venv/bin/activate

## Install PIP requirements
pip install -r requirements.txt 

PIP Requirements:
robotframework
robotframework-seleniumlibrary
